# Personal Portfolio

**[View Live Site](https://labjankiness.github.io/Personal-Portfolio-WIP/)**

A responsive developer portfolio website showcasing cybersecurity, AI/ML, systems programming, and web development projects.

## Pages

- **Home** (`index.html`) — Hero section, tech stack grid, contact info
- **Projects** (`projects.html`) — 7 project cards with category tags (Cybersecurity, AI/ML, Web App, Game Dev, Systems)
- **Blog** (`blog.html`) — Technical write-ups on GPA Calculator, Port Scanner Suite, and MyOS

## Featured Projects

| Project | Category |
|---|---|
| Port Scanner Suite | Cybersecurity |
| Local AI SOC Agent | AI/ML |
| RP GPA Calculator | Web App |
| FPS Protocol | Game Dev |
| MyOS | Systems |
| ZephyrOS | Systems |
| Comms Link Diagnostic Agent | AI/ML |

## Features

- **Dark / Light theme** toggle with preference saved in localStorage
- **Glassmorphism navbar** with backdrop blur effect
- **Card hover animations** — lift and shadow on project cards and skill tags
- **Responsive grid layout** — auto-fit columns (280px min) with 600px mobile breakpoint
- **Active nav states** — current page highlighted in accent color

## Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks)
- Inter font via Google Fonts
- CSS custom properties for dark/light theming
- Smooth scroll navigation

## File Structure

```
Personal Portfolio/
├── index.html       # Home page with hero, tech stack, contact
├── projects.html    # Project showcase with category-tagged cards
├── blog.html        # Technical blog articles
├── css/style.css    # All styling with dark/light theme support

```
